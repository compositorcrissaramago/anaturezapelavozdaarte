Sonificação de Dados

A natureza pela voz da arte

Este projeto propõe a criação de obras sonoras/musicais baseadas na sonificação de dados através de algoritmos de conversão e endereçamento propostos pelo compositor. Dados serão interpretados como valores de referência que servirão de inputs para o sistema, advindos de plataformas específicas (climatempo, dados geográficos, etc.). A ideia é consolidar processos de geração sonora que aproveitem dados reais, captados de fontes digitais, para gerar sons a partir de parâmetros e funções propostos pelo compositor, desta forma estaremos sonificando (materializando em som) os valores processados.

Com o desenvolvimento do sistema atuante no reconhecimento, captura, processamento e endereçamento de dados para o controle de instrumentos virtuais, serão geradas sonoridades que variam tais quais os dados que as alimentam. Esta proposta dialoga com questões do universo musical e artístico e as possibilidades tecnológicas contemporâneas, também revisa conceitos sobre autoria, inspiração, representação sonora, além de aplicar na prática ferramentas de engenharia de áudio, técnicas de composição e desenvolvimento de código computacional, como inteligência artificial, sintetizadores, teoria musical e abordagens artísticas.
